# F1.2 Chọn Template

## Tổng quan

Host chọn template khi tạo event (F1.1) hoặc đổi template cho event đã có. Template có 2 loại: **free** và **premium**. Đổi template trên event đang có sẽ xóa toàn bộ `settings` JSONB → cần confirm trước.

---

## Hai ngữ cảnh sử dụng

| Ngữ cảnh | Route | Hành vi sau chọn |
|---------|-------|-----------------|
| Tạo event mới (F1.1) | `/dashboard/events/create` | Template picker nằm trong form tạo event, giá trị là `template_id` field |
| Đổi template cho event có sẵn | `/dashboard/events/{slug}/template` | Confirm dialog → reset `settings` → reload drag editor |

## Controller

```php
// app/Http/Controllers/Dashboard/EventTemplateController.php
class EventTemplateController extends Controller
{
    public function __construct(private TemplateService $templateService) {}

    public function edit(Event $event): Response
    {
        $this->authorize('update', $event);

        return Inertia::render('Dashboard/Events/TemplateSelect', [
            'event'          => $event->only('id', 'slug', 'title', 'template_id'),
            'templates'      => $this->templateService->listForUser(auth()->user()),
            'currentTemplate'=> $event->template_id,
        ]);
    }

    public function update(UpdateEventTemplateRequest $request, Event $event): RedirectResponse
    {
        $this->authorize('update', $event);

        $this->templateService->changeTemplate(
            auth()->user(),
            $event,
            $request->validated('template_id')
        );

        return redirect()
            ->route('dashboard.events.show', $event->slug)
            ->with('success', __('event.template_changed'));
    }
}
```

## Service

```php
// app/Services/TemplateService.php
class TemplateService
{
    public function __construct(
        private TemplateRepository $templateRepo,
    ) {}

    public function listForUser(User $user): Collection
    {
        $ownedIds = $user->purchasedTemplates()->pluck('template_id')->toArray();

        return Template::active()
            ->get(['id', 'name', 'category', 'thumbnail_path', 'price', 'is_premium'])
            ->map(fn ($t) => array_merge($t->toArray(), [
                'is_owned'     => $t->price === 0 || in_array($t->id, $ownedIds),
                'thumbnail_url'=> Storage::url($t->thumbnail_path),
            ]));
    }

    public function changeTemplate(User $user, Event $event, int $templateId): void
    {
        $template = Template::findOrFail($templateId);

        // Kiểm tra quyền dùng premium template
        if ($template->price > 0) {
            $owned = $user->purchasedTemplates()
                ->where('template_id', $templateId)
                ->exists();

            if (!$owned) {
                throw new TemplateNotPurchasedException($template);
            }
        }

        // Reset settings vì layout mới hoàn toàn khác
        $event->update([
            'template_id' => $templateId,
            'settings'    => [],
        ]);

        // Regenerate OG image với template mới
        GenerateOgImage::dispatch($event->fresh());
    }
}
```

## Vue Component

```vue
<!-- resources/js/Pages/Dashboard/Events/TemplateSelect.vue -->
<script setup>
import { ref, computed } from 'vue'
import { router } from '@inertiajs/vue3'

const props = defineProps({
    event: Object,
    templates: Array,
    currentTemplate: Number,
})

const filter = ref('all')
const previewTemplate = ref(null)
const confirmChange = ref(false)
const pendingTemplateId = ref(null)

const filtered = computed(() => {
    if (filter.value === 'owned') return props.templates.filter(t => t.is_owned)
    if (filter.value === 'all') return props.templates
    return props.templates.filter(t => t.category === filter.value)
})

function selectTemplate(template) {
    if (!template.is_owned) {
        // Redirect sang trang mua (F3.x)
        router.visit(route('payment.template', template.id))
        return
    }

    if (template.id === props.currentTemplate) return

    // Cần confirm vì sẽ reset settings
    pendingTemplateId.value = template.id
    confirmChange.value = true
}

function confirmAndChange() {
    router.patch(
        route('dashboard.events.template.update', props.event.slug),
        { template_id: pendingTemplateId.value },
        { onFinish: () => { confirmChange.value = false } }
    )
}
</script>

<template>
    <DashboardLayout>
        <!-- Filter tabs -->
        <div class="flex gap-2">
            <button v-for="cat in ['all', 'wedding', 'birthday', 'conference', 'owned']"
                    :key="cat"
                    @click="filter = cat"
                    :class="{ 'active': filter === cat }">
                {{ cat }}
            </button>
        </div>

        <!-- Template grid -->
        <div class="grid grid-cols-3 gap-4">
            <div v-for="t in filtered" :key="t.id" class="relative cursor-pointer group"
                 @click="previewTemplate = t">
                <img :src="t.thumbnail_url" class="w-full rounded-lg" />

                <!-- Badges -->
                <span v-if="t.price === 0" class="badge-free">FREE</span>
                <span v-else class="badge-premium">
                    {{ t.is_owned ? 'Đã sở hữu' : t.price.toLocaleString('vi-VN') + 'đ' }}
                </span>

                <!-- Selected indicator -->
                <div v-if="t.id === currentTemplate" class="absolute inset-0 border-2 border-primary rounded-lg" />

                <!-- Hover overlay -->
                <div class="hidden group-hover:flex absolute inset-0 items-center justify-center bg-black/40 rounded-lg">
                    <span class="text-white">Xem thử</span>
                </div>
            </div>
        </div>

        <!-- Full-screen preview modal -->
        <teleport to="body">
            <div v-if="previewTemplate" class="fixed inset-0 z-50 flex">
                <iframe :src="route('template.preview', previewTemplate.id)"
                        class="flex-1 h-full border-0" />
                <div class="w-72 bg-white p-6 flex flex-col gap-4">
                    <h2>{{ previewTemplate.name }}</h2>
                    <p>{{ previewTemplate.price === 0 ? 'Miễn phí' : previewTemplate.price.toLocaleString('vi-VN') + 'đ' }}</p>

                    <button v-if="previewTemplate.is_owned"
                            @click="selectTemplate(previewTemplate)">
                        Chọn template này
                    </button>
                    <button v-else
                            @click="selectTemplate(previewTemplate)">
                        Mua — {{ previewTemplate.price.toLocaleString('vi-VN') }}đ
                    </button>

                    <button @click="previewTemplate = null">Đóng</button>
                </div>
            </div>
        </teleport>

        <!-- Confirm reset settings dialog -->
        <ConfirmDialog v-model="confirmChange"
                       title="Đổi template?"
                       description="Đổi template sẽ xóa toàn bộ thiết lập vị trí/font/màu hiện tại. Hành động này không thể hoàn tác."
                       confirm-label="Đổi template"
                       @confirm="confirmAndChange" />
    </DashboardLayout>
</template>
```

## Exceptions

```php
// app/Exceptions/TemplateNotPurchasedException.php
class TemplateNotPurchasedException extends Exception
{
    public function __construct(Template $template)
    {
        parent::__construct(__('template.not_purchased', ['name' => $template->name]));
    }

    public function render(): RedirectResponse
    {
        return back()->withErrors(['template' => $this->getMessage()]);
    }
}
```

## Edge Cases

| Tình huống | Xử lý |
|-----------|-------|
| Chọn lại template đang dùng | Frontend disable nút, không gửi request |
| Template premium chưa mua | Redirect sang trang thanh toán (F3.x) |
| Template bị deactivate sau khi user load trang | Validation `is_active = true` fail → lỗi |
| Đổi template → confirm → mạng chậm | `router.patch` xử lý loading state tự động |
| Preview iframe load chậm | Skeleton placeholder trong iframe wrapper |
| `settings` có data phức tạp rồi mới đổi | Reset `settings = []` — đã confirm trước |
