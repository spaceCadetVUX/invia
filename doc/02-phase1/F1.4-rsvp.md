# F1.4 RSVP

## Tổng quan

Khách xác nhận tham dự qua form Blade (không cần tài khoản). Hai chế độ: **token** (link cá nhân `?t=`) và **anonymous** (link chung). Sau submit → gửi email xác nhận cho khách + push notification cho host.

---

## Hai chế độ RSVP

| Chế độ | Điều kiện | Behavior |
|--------|-----------|----------|
| **Token** | URL có `?t={token}` hợp lệ | Pre-fill tên khách, không hỏi tên. Update record `rsvp` của guest đó. |
| **Anonymous** | Không có token hoặc token sai | Hỏi tên + email (optional). Tạo guest mới `source=self_register` + record `rsvp`. |

## Controller

```php
// app/Http/Controllers/RsvpController.php
class RsvpController extends Controller
{
    public function __construct(private RsvpService $rsvpService) {}

    public function form(string $slug, Request $request): View
    {
        $event = $this->rsvpService->resolvePublishedEvent($slug);
        $guest = $this->rsvpService->resolveGuest($request->query('t'), $event);

        abort_if(!$event->rsvp_enabled, 404);

        // Existing RSVP nếu token hợp lệ
        $existingRsvp = $guest
            ? $guest->rsvp()->where('event_id', $event->id)->first()
            : null;

        return view('invitation.rsvp', compact('event', 'guest', 'existingRsvp'));
    }

    public function store(StoreRsvpRequest $request, string $slug): RedirectResponse
    {
        $event = $this->rsvpService->resolvePublishedEvent($slug);

        abort_if(!$event->rsvp_enabled, 403);

        $rsvp = $this->rsvpService->handleRsvp(
            event: $event,
            data: $request->validated(),
            token: $request->query('t'),
            ip: $request->ip(),
        );

        return redirect()
            ->route('invitation.rsvp.success', $slug)
            ->with('rsvp_status', $rsvp->status);
    }

    public function success(string $slug, Request $request): View
    {
        $event = Event::where('slug', $slug)->where('status', 'published')->firstOrFail();

        return view('invitation.rsvp-success', [
            'event'       => $event,
            'rsvp_status' => session('rsvp_status', 'yes'),
        ]);
    }
}
```

## Request Validation

```php
// app/Http/Requests/StoreRsvpRequest.php
class StoreRsvpRequest extends FormRequest
{
    public function rules(): array
    {
        // Anonymous: name required. Token mode: name không cần (lấy từ guest record)
        $nameRule = $this->query('t') ? ['nullable', 'string', 'max:100']
                                       : ['required', 'string', 'max:100'];

        return [
            'name'     => $nameRule,
            'email'    => ['nullable', 'email', 'max:150'],
            'status'   => ['required', Rule::in(['yes', 'no', 'maybe'])],
            'plus_one' => ['nullable', 'integer', 'min:0', 'max:10'],
            'dietary'  => ['nullable', 'string', 'max:200'],
        ];
    }
}
```

## Service

```php
// app/Services/RsvpService.php
class RsvpService
{
    public function __construct(
        private RsvpRepository $rsvpRepo,
        private GuestRepository $guestRepo,
        private NotificationService $notifService,
    ) {}

    public function handleRsvp(Event $event, array $data, ?string $token, string $ip): Rsvp
    {
        // Resolve trước — scope theo event_id để tránh token cross-event
        $guest = $token
            ? Guest::where('token', $token)->where('event_id', $event->id)->first()
            : null;

        if ($guest) {
            return $this->handleTokenRsvp($event, $data, $guest);
        }

        // Token sai/không tồn tại/thuộc event khác → fallback anonymous
        return $this->handleAnonymousRsvp($event, $data, $ip);
    }

    private function handleTokenRsvp(Event $event, array $data, Guest $guest): Rsvp
    {

        $rsvp = $this->rsvpRepo->upsert([
            'guest_id'  => $guest->id,
            'event_id'  => $event->id,
            'status'    => $data['status'],
            'plus_one'  => $data['plus_one'] ?? 0,
            'dietary'   => $data['dietary'] ?? null,
        ]);

        $this->afterRsvp($event, $guest, $rsvp, isNew: $rsvp->wasRecentlyCreated);

        return $rsvp;
    }

    private function handleAnonymousRsvp(Event $event, array $data, string $ip): Rsvp
    {
        // Tạo guest mới source=self_register
        $guest = $this->guestRepo->create([
            'event_id' => $event->id,
            'name'     => $data['name'],
            'email'    => $data['email'] ?? null,
            'token'    => Str::random(32),
            'source'   => 'self_register',
        ]);

        $rsvp = $this->rsvpRepo->create([
            'guest_id' => $guest->id,
            'event_id' => $event->id,
            'status'   => $data['status'],
            'plus_one' => $data['plus_one'] ?? 0,
            'dietary'  => $data['dietary'] ?? null,
        ]);

        $this->afterRsvp($event, $guest, $rsvp, isNew: true);

        return $rsvp;
    }

    private function afterRsvp(Event $event, Guest $guest, Rsvp $rsvp, bool $isNew): void
    {
        // Email xác nhận cho khách (nếu có email)
        if ($guest->email) {
            SendRsvpConfirmation::dispatch($guest, $event, $rsvp);
        }

        // Notification cho host (chỉ khi RSVP mới, không spam khi update)
        if ($isNew) {
            $this->notifService->notifyHost($event, 'rsvp', [
                'guest_name'  => $guest->name,
                'rsvp_status' => $rsvp->status,
                'guest_id'    => $guest->id,
                'event_id'    => $event->id,
            ]);
        }
    }

    public function resolvePublishedEvent(string $slug): Event
    {
        return Event::where('slug', $slug)
            ->where('status', 'published')
            ->firstOrFail();
    }

    public function resolveGuest(?string $token, Event $event): ?Guest
    {
        if (!$token) return null;
        return Guest::where('token', $token)->where('event_id', $event->id)->first();
    }
}
```

## Repository

```php
// app/Repositories/RsvpRepository.php
class RsvpRepository
{
    public function upsert(array $data): Rsvp
    {
        // updateOrCreate để token mode có thể update RSVP đã có
        return Rsvp::updateOrCreate(
            ['guest_id' => $data['guest_id'], 'event_id' => $data['event_id']],
            $data,
        );
    }

    public function create(array $data): Rsvp
    {
        return Rsvp::create($data);
    }
}
```

## Rate Limiting

```php
// app/Http/Middleware/ThrottleRsvp.php — hoặc config trong RouteServiceProvider
RateLimiter::for('rsvp', function (Request $request) {
    return [
        // Giới hạn per IP: 10 lần / 10 phút
        Limit::perMinutes(10, 10)->by($request->ip()),
        // Giới hạn per event: 200 lần / giờ (tránh spam event nhỏ)
        Limit::perHour(200)->by('event:' . $request->route('slug')),
    ];
});
```

## Queue Jobs

### SendRsvpConfirmation

```php
// app/Jobs/SendRsvpConfirmation.php
class SendRsvpConfirmation implements ShouldQueue
{
    public function __construct(
        public Guest $guest,
        public Event $event,
        public Rsvp $rsvp,
    ) {}

    public function handle(): void
    {
        Mail::to($this->guest->email)->send(new RsvpConfirmationMail(
            guest: $this->guest,
            event: $this->event,
            rsvp:  $this->rsvp,
        ));
    }
}
```

## Edge Cases

| Tình huống | Xử lý |
|-----------|-------|
| Token hợp lệ, đã RSVP trước | `updateOrCreate` → update record cũ, không tạo mới. Show "đã xác nhận" notice trên form |
| Token sai → anonymous mode | `resolveGuest()` trả null → show form đầy đủ |
| `rsvp_enabled = false` trên event | `abort(404)` trên GET, `abort(403)` trên POST |
| IP spam > 10 lần / 10 phút | `throttle:rsvp` → HTTP 429 |
| Anonymous không nhập email | OK — email xác nhận không gửi, RSVP vẫn lưu |
| Token thuộc event khác | `handleTokenRsvp` dùng `$event->id` khi upsert → không ghi chéo event |
| `plus_one` null | Default 0 |
| Host tắt RSVP sau khi khách đã mở form | POST bị `abort(403)`, GET vẫn thấy form nhưng POST fail |
| Queue down → email không gửi | Job retry tự động, RSVP đã lưu DB — không mất data |

- [ ] `RateLimiter::for('rsvp')` định nghĩa trong `AppServiceProvider`
- [ ] `rsvp_enabled` field có trong `events` table (migration)
- [ ] `upsert` dùng `updateOrCreate` — token mode không tạo duplicate
- [ ] Anonymous tạo guest mới `source = self_register` + random token
- [ ] `SendRsvpConfirmation` chỉ dispatch khi `guest->email` không null
- [ ] Host notification chỉ khi `isNew = true` (không spam khi update)
- [ ] `?t=` truyền lại qua action URL của form
- [ ] Form pre-fill từ `$existingRsvp` khi token mode
- [ ] Success page đọc `session('rsvp_status')` để show message đúng
