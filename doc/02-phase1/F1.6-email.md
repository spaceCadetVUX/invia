# F1.6 Gửi Email Thiệp

## Tổng quan

Host gửi link thiệp đến danh sách khách mời qua email. Mỗi khách nhận link cá nhân (`?t={token}`) hoặc link chung nếu không có token. Gửi bulk qua queue job, không block UI. Hỗ trợ resend cho khách chưa mở email.

---

## Flow

```
Dashboard /dashboard/events/{slug}/send
  → Host chọn: Gửi tất cả / Gửi chưa gửi / Chọn thủ công
  → Confirm số lượng → POST
    → Queue: SendInvitationEmail × N jobs
      → Mỗi job gửi 1 email qua Resend.com
      → Update guests.email_sent_at
    → Host xem progress qua polling dashboard
```

## Controller

```php
// app/Http/Controllers/Dashboard/SendInvitationController.php
class SendInvitationController extends Controller
{
    public function __construct(private SendInvitationService $sendService) {}

    public function index(Event $event): Response
    {
        $this->authorize('update', $event);

        return Inertia::render('Dashboard/Events/Send', [
            'event'      => $event->only('id', 'slug', 'title', 'status'),
            'stats'      => $this->sendService->getSendStats($event),
        ]);
    }

    public function send(SendInvitationRequest $request, Event $event): JsonResponse
    {
        $this->authorize('update', $event);

        abort_if($event->status !== 'published', 422, __('send.event_not_published'));

        $count = $this->sendService->dispatch(
            event:     $event,
            mode:      $request->validated('mode'),
            guestIds:  $request->validated('guest_ids'),
        );

        return response()->json([
            'queued' => $count,
            'message'=> __('send.queued', ['count' => $count]),
        ]);
    }

    public function progress(Event $event): JsonResponse
    {
        $this->authorize('update', $event);

        return response()->json($this->sendService->getSendStats($event));
    }
}
```

```php
                ->where('event_id', fn () => request()->route('event')?->id)],
        ];
```

## Service

```php
// app/Services/SendInvitationService.php
class SendInvitationService
{
    public function dispatch(Event $event, string $mode, ?array $guestIds): int
    {
        $guests = $this->resolveGuests($event, $mode, $guestIds);

        // Lọc bỏ: không có email, đã unsubscribe
        $eligible = $guests->filter(fn ($g) =>
            $g->email &&
            !$g->unsubscribed_at
        );

        foreach ($eligible as $guest) {
            SendInvitationEmail::dispatch($event, $guest)
                ->onQueue('emails');
        }

        return $eligible->count();
    }

    public function getSendStats(Event $event): array
    {
        $base         = Guest::where('event_id', $event->id)->whereNotNull('email');
        $total        = $base->count();
        $sent         = (clone $base)->whereNotNull('email_sent_at')->count();
        $unsubscribed = (clone $base)->whereNotNull('unsubscribed_at')->count();
        // unsent = chưa gửi VÀ chưa unsubscribe (số thực sự sẽ được gửi)
        $unsent       = (clone $base)->whereNull('email_sent_at')->whereNull('unsubscribed_at')->count();

        return compact('total', 'sent', 'unsent', 'unsubscribed');
    }

    private function resolveGuests(Event $event, string $mode, ?array $guestIds): Collection
    {
        $query = Guest::where('event_id', $event->id);

        return match ($mode) {
            'all'    => $query->get(),
            'unsent' => $query->whereNull('email_sent_at')->get(),
            'manual' => $query->whereIn('id', $guestIds ?? [])->get(),
        };
    }
}
```

## Queue Job

```php
// app/Jobs/SendInvitationEmail.php
class SendInvitationEmail implements ShouldQueue
{
    use Dispatchable, InteractsWithQueue, Queueable, SerializesModels;

    public int $tries = 3;
    public int $backoff = 60;   // retry sau 60s

    public function __construct(
        public Event $event,
        public Guest $guest,
    ) {}

    public function handle(): void
    {
        // Double-check unsubscribe trước khi gửi (có thể unsubscribe sau khi job đã queue)
        if ($this->guest->unsubscribed_at) {
            return;
        }

        Mail::to($this->guest->email)
            ->send(new InvitationMail($this->event, $this->guest));

        // Ghi nhận đã gửi
        $this->guest->update(['email_sent_at' => now()]);
    }

    public function failed(\Throwable $e): void
    {
        // Log nhưng không crash — job khác vẫn chạy
        Log::error("SendInvitationEmail failed: guest {$this->guest->id}", [
            'error' => $e->getMessage(),
        ]);
    }
}
```

## Mailable

```php
// app/Mail/InvitationMail.php
class InvitationMail extends Mailable
{
    use SerializesModels;

    public function __construct(
        public Event $event,
        public Guest $guest,
    ) {}

    public function envelope(): Envelope
    {
        return new Envelope(
            subject: __('email.invitation_subject', ['title' => $this->event->title]),
        );
    }

    public function content(): Content
    {
        return new Content(view: 'emails.invitation');
    }

    public function headers(): Headers
    {
        // List-Unsubscribe header — bắt buộc cho email deliverability
        $unsubUrl = route('unsubscribe', $this->guest->token);

        return new Headers(
            text: [
                'List-Unsubscribe'      => "<{$unsubUrl}>",
                'List-Unsubscribe-Post' => 'List-Unsubscribe=One-Click',
            ]
        );
    }
}
```

```php
// app/Http/Controllers/UnsubscribeController.php
class UnsubscribeController extends Controller
{
    public function handle(string $token): View
    {
        $guest = Guest::where('token', $token)->firstOrFail();

        if (!$guest->unsubscribed_at) {
            $guest->update(['unsubscribed_at' => now()]);
        }

        return view('invitation.unsubscribed', ['guest' => $guest]);
    }
}
```

## Resend Logic

Host có thể resend cho:
- **Chưa gửi** (`email_sent_at IS NULL`) — mode `unsent`
- **Chọn thủ công** — mode `manual`, tích từng guest trong bảng

Không có "resend tất cả" để tránh spam. Mode `all` chỉ gửi lần đầu (vẫn lọc qua `email_sent_at` ở tầng UX — backend không block).

## Vue Component (Dashboard)

```vue
<!-- resources/js/Pages/Dashboard/Events/Send.vue -->
<script setup>
import { ref, onMounted, onUnmounted } from 'vue'
import { router } from '@inertiajs/vue3'
import axios from 'axios'

const props = defineProps({
    event: Object,
    stats: Object,
})

const stats = ref(props.stats)
const sending = ref(false)
const mode = ref('unsent')
const selectedIds = ref([])
let pollTimer = null

async function send() {
    sending.value = true
    const res = await axios.post(route('dashboard.events.send.store', props.event.slug), {
        mode: mode.value,
        guest_ids: mode.value === 'manual' ? selectedIds.value : undefined,
    })
    alert(res.data.message)
    startPolling()
}

function startPolling() {
    pollTimer = setInterval(async () => {
        const res = await axios.get(route('dashboard.events.send.progress', props.event.slug))
        stats.value = res.data
        // Dừng khi unsent = 0
        if (res.data.unsent === 0) stopPolling()
    }, 3000)
}

function stopPolling() {
    clearInterval(pollTimer)
    sending.value = false
}

onUnmounted(stopPolling)
</script>

<template>
    <DashboardLayout>
        <!-- Stats row -->
        <div class="grid grid-cols-4 gap-4">
            <StatCard label="Tổng có email"    :value="stats.total" />
            <StatCard label="Đã gửi"           :value="stats.sent" />
            <StatCard label="Chưa gửi"         :value="stats.unsent" />
            <StatCard label="Hủy nhận"         :value="stats.unsubscribed" />
        </div>

        <!-- Send mode selector -->
        <div class="flex gap-4 mt-6">
            <label><input type="radio" v-model="mode" value="all">    Gửi tất cả</label>
            <label><input type="radio" v-model="mode" value="unsent"> Chỉ chưa gửi</label>
            <label><input type="radio" v-model="mode" value="manual"> Chọn thủ công</label>
        </div>

        <button @click="send" :disabled="sending || event.status !== 'published'" class="btn-primary mt-4">
            {{ sending ? 'Đang gửi...' : 'Gửi thiệp' }}
        </button>

        <p v-if="event.status !== 'published'" class="text-warning mt-2">
            {{ $t('send.publish_first') }}
        </p>
    </DashboardLayout>
</template>
```

## Edge Cases

| Tình huống | Xử lý |
|-----------|-------|
| Khách không có email | Lọc trước khi dispatch — không queue job |
| Khách đã unsubscribe | Lọc trong `dispatch()` + double-check trong job `handle()` |
| Event chưa publish | `abort(422)` với message rõ ràng |
| Job fail sau 3 lần retry | `failed()` log error, job khác vẫn chạy độc lập |
| Resend.com rate limit | Queue `backoff=60` retry sau 60s, tránh burst |
| Host gửi 2 lần nhanh | Mode `unsent` — lần 2 không có ai vì `email_sent_at` đã set |
| `guest_ids` trong manual có guest của event khác | `Rule::exists('guests', 'id')` không check event_id — cần thêm scope |
| 1,000+ khách | Queue `emails` xử lý dần, không block — estimate ~30 phút với 10 workers |
| Unsubscribe token sai/giả | `firstOrFail()` → 404 |
