# F1.3 Trang Thiệp Public

## Tổng quan

Trang `/thiep/{slug}` render server-side bằng **Blade** — bắt buộc để crawler Facebook/Zalo đọc được OG tags khi share link. Hỗ trợ 2 chế độ: link chung (không cá nhân hóa) và link cá nhân (`?t={token}`, hiển thị tên khách).

---

## Hai chế độ truy cập

| URL | Chế độ | Hiển thị |
|-----|--------|---------|
| `/thiep/{slug}` | Link chung | Thiệp đầy đủ, không có tên khách |
| `/thiep/{slug}?t={token}` | Link cá nhân | Thêm "Kính mời **{tên khách}**" trên thiệp |

## Controller

```php
// app/Http/Controllers/InvitationController.php
class InvitationController extends Controller
{
    public function __construct(private InvitationService $invitationService) {}

    public function show(string $slug, Request $request): View|Response
    {
        // Cache 5 phút — invalidate khi host edit/publish (Cache::forget("invitation:{$slug}"))
        $event = Cache::remember("invitation:{$slug}", 300, fn () =>
            $this->invitationService->resolveEvent($slug)
        );
        $guest = $this->invitationService->resolveGuest($request->query('t'), $event);

        // Track lượt xem (không block render)
        TrackEventView::dispatchAfterResponse($event->id, $guest?->id);

        return view("templates.{$event->template->blade_file}.index", [
            'event'    => $event,
            'guest'    => $guest,
            'music'    => $this->invitationService->resolveMusic($event),
            'ogMeta'   => $this->invitationService->buildOgMeta($event),
        ]);
    }

    public function calendar(string $slug): Response
    {
        $event = $this->invitationService->resolveEvent($slug);

        $ics = $this->invitationService->generateIcs($event);

        return response($ics, 200, [
            'Content-Type'        => 'text/calendar; charset=UTF-8',
            'Content-Disposition' => 'attachment; filename="su-kien.ics"',
        ]);
    }
}
```

## Service

```php
// app/Services/InvitationService.php
class InvitationService
{
    public function resolveEvent(string $slug): Event
    {
        $event = Event::with('template')
            ->where('slug', $slug)
            ->where('status', 'published')
            ->first();

        if (!$event) {
            abort(404);
        }

        // Hết hạn → trang thông báo riêng (không 404)
        if ($event->expires_at && $event->expires_at->isPast()) {
            abort(response()->view('errors.invitation-expired', ['event' => $event], 410));
        }

        return $event;
    }

    public function resolveGuest(?string $token, Event $event): ?Guest
    {
        if (!$token) return null;

        // Scope theo event_id — tránh token của event khác hiển thị tên sai
        return Guest::where('token', $token)->where('event_id', $event->id)->first();
        // Không abort nếu token sai — chỉ hiện thiệp không cá nhân hóa
    }

    public function resolveMusic(Event $event): array
    {
        return match ($event->music_type) {
            'library'    => $this->libraryMusic($event),
            'soundcloud' => ['type' => 'soundcloud', 'url' => $event->music_source],
            'upload'     => ['type' => 'upload',     'url' => Storage::url($event->music_source)],
            default      => ['type' => 'none'],
        };
    }

    public function buildOgMeta(Event $event): array
    {
        $image = $event->og_image_path
            ? Storage::url($event->og_image_path)
            : asset('images/og-default.jpg');

        return [
            'title'       => $event->title . ' — Thiệp mời',
            'description' => "Trân trọng kính mời bạn tới {$event->venue_name} vào ngày {$event->event_date->format('d/m/Y')}.",
            'url'         => route('invitation.show', $event->slug),
            'image'       => $image,
            'image_width' => 1200,
            'image_height'=> 630,
        ];
    }

    public function generateIcs(Event $event): string
    {
        $start = $event->event_date->format('Ymd') . ($event->event_time ? 'T' . str_replace(':', '', $event->event_time) . '00' : '');
        $uid   = $event->slug . '@invia.vn';

        return implode("\r\n", [
            'BEGIN:VCALENDAR',
            'VERSION:2.0',
            'PRODID:-//Invia.vn//Thiệp mời//VI',
            'BEGIN:VEVENT',
            "UID:{$uid}",
            "DTSTART:{$start}",
            "SUMMARY:{$event->title}",
            "LOCATION:{$event->venue_name}, {$event->venue_address}",
            "DESCRIPTION:" . route('invitation.show', $event->slug),
            'END:VEVENT',
            'END:VCALENDAR',
        ]);
    }

    private function libraryMusic(Event $event): array
    {
        if (!$event->music_source) return ['type' => 'none'];

        $track = MusicLibrary::find((int) $event->music_source);

        return $track
            ? ['type' => 'library', 'url' => Storage::url($track->file_path), 'title' => $track->title]
            : ['type' => 'none'];
    }
}
```

## Nhạc nền — 3 loại

```blade
{{-- resources/views/partials/invitation-music.blade.php --}}
@if ($music['type'] === 'none')
    {{-- Không có nhạc --}}

@elseif ($music['type'] === 'library' || $music['type'] === 'upload')
    {{-- HTML5 Audio — autoplay với mute ban đầu để pass browser policy --}}
    <audio id="bgMusic" src="{{ $music['url'] }}" loop preload="auto" style="display:none"></audio>
    <button id="musicToggle" aria-label="Bật/tắt nhạc" class="music-toggle">
        <span class="icon-music-on">♪</span>
    </button>
    <script>
    (function () {
        const audio  = document.getElementById('bgMusic');
        const toggle = document.getElementById('musicToggle');
        // Autoplay sau interaction đầu tiên (browser policy)
        document.addEventListener('click', function startMusic() {
            audio.play().catch(() => {});
            document.removeEventListener('click', startMusic);
        }, { once: true });
        toggle.addEventListener('click', () => {
            audio.paused ? audio.play() : audio.pause();
        });
    })();
    </script>

@elseif ($music['type'] === 'soundcloud')
    {{-- SoundCloud Widget embed --}}
    <div class="soundcloud-player" style="display:none">
        <iframe
            src="https://w.soundcloud.com/player/?url={{ urlencode($music['url']) }}&auto_play=true&hide_related=true&show_comments=false&show_user=false&show_reposts=false"
            width="100%" height="166" frameborder="0" allow="autoplay">
        </iframe>
    </div>
@endif
```

**Lý do autoplay behavior:**
- Chrome/Safari block autoplay có âm thanh trừ khi user đã tương tác với trang
- Giải pháp: start audio sau `click` đầu tiên (scroll, tap vào thiệp đều kích hoạt)
- SoundCloud widget có `auto_play=true` nhưng browser cũng áp dụng rule tương tự

## Queue Job — TrackEventView

```php
// app/Jobs/TrackEventView.php
class TrackEventView implements ShouldQueue
{
    public function __construct(
        public int $eventId,
        public ?int $guestId,
    ) {}

    public function handle(): void
    {
        // Increment event view counter — không block response
        Event::where('id', $this->eventId)->increment('view_count');
    }
}
```

## Cache Strategy

Cache đã được tích hợp trực tiếp vào `show()`. Key: `"invitation:{slug}"`, TTL: 300s.

**Invalidate** trong `EventService` khi host publish hoặc edit event:

```php
// app/Services/EventService.php
Cache::forget("invitation:{$event->slug}");
```

## Edge Cases

| Tình huống | Xử lý |
|-----------|-------|
| Event chưa publish (draft) | `resolveEvent()` check `status = published` → 404 |
| Event hết hạn | HTTP 410 + trang "Thiệp đã hết hạn" (không 404) |
| Token `?t=` sai/giả | Không abort — render thiệp không tên khách |
| Token thuộc event khác | Không hiển thị tên sai — resolve guest sau khi có event |
| Template bị deactivate sau khi publish | Vẫn render được — view file vẫn tồn tại trên disk |
| `og_image_path` null (job chưa chạy xong) | Fallback về `og-default.jpg` |
| SoundCloud track bị gỡ | Widget hiện lỗi, thiệp vẫn load bình thường |
| `music_source` null khi type = library | `resolveMusic()` trả `['type' => 'none']` |
| Autoplay bị browser block | Audio chờ `click` đầu tiên — không crash |
| `.ics` khi không có `event_time` | DTSTART format ngày không có giờ (`YYYYMMDD`) |

## Checklist

- [ ] Route `/thiep/{slug}` không có middleware auth
- [ ] `resolveEvent()` check `status = published` + `expires_at`
- [ ] `resolveGuest()` không abort khi token sai — silent fallback
- [ ] OG tags render đầy đủ trong `<head>` — test bằng Facebook Debugger
- [ ] `<meta name="robots" content="noindex">` có trong layout
- [ ] Cache 5 phút cho event, invalidate khi edit/publish
- [ ] `TrackEventView::dispatchAfterResponse()` — không block render
- [ ] `.ics` download đúng MIME type `text/calendar`
- [ ] Nhạc library/upload: audio start sau user click đầu tiên
- [ ] SoundCloud embed dùng `https://w.soundcloud.com/player/` URL
- [ ] Template CSS/JS load với version query string (`?v={version}`)
- [ ] Trang expired trả HTTP 410 (không 404)
- [ ] `robots.txt` disallow `/thiep/`
