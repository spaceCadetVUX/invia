# F1.1 Tạo Event

## Tổng quan

Host tạo event mới từ dashboard. Form nhập thông tin cơ bản → server validate → tạo record → redirect sang trang detail event.

---

## UI Flow

```
/dashboard                →  nút "Tạo thiệp mới"
  → /dashboard/events/create  (GET, Inertia page)
    → form submit (POST /dashboard/events)
      → success: redirect /dashboard/events/{slug}
      → error:   trả về form với validation errors
```

## Form Fields

| Field | Type | Required | Validation |
|-------|------|----------|-----------|
| `title` | text | ✓ | 3–120 chars, string |
| `event_date` | date | ✓ | date, after_or_equal: today |
| `event_time` | time | ✗ | nullable, date_format: H:i |
| `venue_name` | text | ✗ | nullable, max:150 |
| `venue_address` | text | ✗ | nullable, max:300 |
| `event_type` | enum | ✓ | in: wedding, birthday, conference, party, other |
| `template_id` | FK | ✓ | exists:templates,id, template must be active |
| `language` | enum | ✗ | default: vi, in: vi (MVP — i18n-ready) |

**Notes:**
- `event_date` trong quá khứ → vẫn cho phép tạo nhưng show warning: "Ngày tổ chức đã qua, bạn có muốn tiếp tục?"
- `title` chỉ có emoji/ký tự đặc biệt → slug fallback: `event-{random6}`

```php
            'language'      => ['nullable', Rule::in(['vi'])],
        ];
    }
}
```

## Architecture

### Controller (thin)

```php
// app/Http/Controllers/Dashboard/EventController.php
class EventController extends Controller
{
    public function __construct(
        private EventService $eventService,
        private TemplateService $templateService,
    ) {}

    public function create(): Response
    {
        return Inertia::render('Dashboard/Events/Create', [
            'templates'  => $this->templateService->listForUser(auth()->user()),
            'eventTypes' => EventType::cases(),
        ]);
    }

    public function store(StoreEventRequest $request): RedirectResponse
    {
        $event = $this->eventService->createEvent(
            auth()->user(),
            $request->validated()
        );

        return redirect()
            ->route('dashboard.events.show', $event->slug)
            ->with('success', __('event.created'));
    }
}
```

### Service

```php
// app/Services/EventService.php
class EventService
{
    public function __construct(
        private EventRepository $eventRepo,
        private TemplateRepository $templateRepo,
    ) {}

    public function createEvent(User $user, array $data): Event
    {
        $this->checkEventQuota($user);

        $slug = $this->generateSlug($data['title']);

        $event = $this->eventRepo->create([
            'user_id'       => $user->id,
            'template_id'   => $data['template_id'],
            'title'         => $data['title'],
            'slug'          => $slug,
            'event_type'    => $data['event_type'],
            'event_date'    => $data['event_date'],
            'event_time'    => $data['event_time'] ?? null,
            'venue_name'    => $data['venue_name'] ?? null,
            'venue_address' => $data['venue_address'] ?? null,
            'language'      => $data['language'] ?? 'vi',
            'settings'      => [],          // JSONB, empty object
            'status'        => 'draft',
            'expires_at'    => now()->addYear(),
        ]);

        GenerateOgImage::dispatch($event);

        return $event;
    }

    private function checkEventQuota(User $user): void
    {
        $plan = $user->plan ?? 'free';
        $limit = config("plans.{$plan}.max_events", 1);
        $count = $this->eventRepo->countActive($user->id);

        if ($count >= $limit) {
            throw new EventQuotaExceededException($plan, $limit);
        }
    }

    private function generateSlug(string $title): string
    {
        $base = Str::slug($title);

        // Fallback nếu title toàn emoji/ký tự không slug được
        if (empty($base)) {
            $base = 'event';
        }

        for ($i = 0; $i < 5; $i++) {
            $candidate = $i === 0 ? $base : $base . '-' . Str::random(6);

            if (!$this->eventRepo->slugExists($candidate)) {
                return $candidate;
            }
        }

        // Last resort sau 5 lần collision
        return $base . '-' . Str::random(8);
    }
}
```

### Repository

```php
// app/Repositories/EventRepository.php
class EventRepository
{
    public function create(array $data): Event
    {
        return Event::create($data);
    }

    public function countActive(int $userId): int
    {
        return Event::where('user_id', $userId)
            ->whereIn('status', ['draft', 'published'])
            ->count();
    }

    public function slugExists(string $slug): bool
    {
        return Event::where('slug', $slug)->exists();
    }
}
```

## Exceptions

```php
// app/Exceptions/EventQuotaExceededException.php
class EventQuotaExceededException extends Exception
{
    public function __construct(string $plan, int $limit)
    {
        parent::__construct(__('event.quota_exceeded', ['plan' => $plan, 'limit' => $limit]));
    }

    public function render(): JsonResponse|RedirectResponse
    {
        return back()->withErrors(['quota' => $this->getMessage()]);
    }
}
```

## Edge Cases

| Tình huống | Xử lý |
|-----------|-------|
| Host free đã có 1 event đang active | Throw `EventQuotaExceededException`, redirect back với error |
| `event_date` trong quá khứ | Vẫn tạo được, show warning banner trên Vue form |
| Title toàn emoji (e.g. "🎉🥳") | `Str::slug()` trả về `""` → dùng base `"event"` + random suffix |
| Slug collision lần 1–4 | Thêm `-{random6}` suffix, retry |
| Slug collision cả 5 lần | `{base}-{random8}` (xác suất cực thấp) |
| Double submit (user click 2 lần) | `form.processing` disable button sau lần đầu |
| `template_id` của template bị deactivate | Validation `is_active = true` fail → lỗi field |
| `GenerateOgImage` job fail | Silent retry 3 lần (queue default), event vẫn tồn tại bình thường |
