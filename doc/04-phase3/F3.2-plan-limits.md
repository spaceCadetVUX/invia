## Tổng quan

`PlanService` là nguồn sự thật duy nhất cho quota và feature flags. Mọi controller cần check giới hạn đều gọi qua service này — không check inline. Khi vượt hạn, throw exception chuẩn → handler render JSON 422 + `error_code` để Vue hiển thị UpgradeCTA đúng ngữ cảnh.

class QuotaExceededException extends \RuntimeException
{
    public function __construct(
        public readonly string $resource,    // 'guests'
        public readonly int    $available,   // slots còn lại
        public readonly string $currentPlan,
    ) {
        parent::__construct("Quota exceeded: {$resource}");
    }
}

// app/Exceptions/PlanFeatureException.php
class PlanFeatureException extends \RuntimeException
{
    public function __construct(
        public readonly string $feature,     // 'email' | 'export' | 'table'
        public readonly string $currentPlan,
    ) {
        parent::__construct("Feature not available: {$feature} on plan {$currentPlan}");
    }
}

## PlanService

```php
// app/Services/PlanService.php
class PlanService
{
    // ── Quota info ─────────────────────────────────────────
    public function getQuota(Event $event): array
    {
        $cfg = config("plans.{$event->plan}", config('plans.free'));

        $maxGuests     = ($cfg['guests'] ?? 0) + $event->extra_guests;
        $currentGuests = Guest::where('event_id', $event->id)->count();

        return [
            'plan'           => $event->plan,
            'max_guests'     => $maxGuests,
            'current_guests' => $currentGuests,
            'remaining'      => max(0, $maxGuests - $currentGuests),
            'can_email'      => (bool) ($cfg['email']  ?? false),
            'can_export'     => (bool) ($cfg['export'] ?? false),
            'can_table'      => (bool) ($cfg['table']  ?? false),
        ];
    }

    // ── Assertions — throw nếu không đủ điều kiện ─────────
    public function assertCanAddGuests(Event $event, int $count = 1): void
    {
        $quota = $this->getQuota($event);

        if ($quota['remaining'] < $count) {
            throw new QuotaExceededException('guests', $quota['remaining'], $event->plan);
        }
    }

    public function assertFeature(Event $event, string $feature): void
    {
        $quota = $this->getQuota($event);

        $allowed = match ($feature) {
            'email'  => $quota['can_email'],
            'export' => $quota['can_export'],
            'table'  => $quota['can_table'],
            default  => false,
        };

        if (!$allowed) {
            throw new PlanFeatureException($feature, $event->plan);
        }
    }

    // ── Event expiry ────────────────────────────────────────
    public function isExpired(Event $event): bool
    {
        if (!$event->expires_at) return false;
        return $event->expires_at->isPast();
    }

    public function computeExpiresAt(string $plan): ?\Carbon\Carbon
    {
        $years = config("plans.{$plan}.storage_years");
        return $years ? now()->addYears($years) : null;   // null = vĩnh viễn
    }
}
```

## Middleware — Block expired events

```php
// app/Http/Middleware/EnsureEventNotExpired.php
class EnsureEventNotExpired
{
    // Payment routes phải exempt — host cần vào đây để reactivate event hết hạn
    private array $except = [
        'dashboard.events.upgrade',
        'dashboard.events.payment.create',
        'dashboard.events.payment.return',
        'dashboard.events.payment.cancel',
        'dashboard.events.payment.coupon-preview',
    ];

    public function handle(Request $request, Closure $next): mixed
    {
        $event = $request->route('event');

        if ($event && app(PlanService::class)->isExpired($event)) {
            // Cho phép vào trang upgrade/payment dù event đã hết hạn
            $routeName = $request->route()->getName();
            if (in_array($routeName, $this->except)) {
                return $next($request);
            }

            if ($request->expectsJson() || $request->header('X-Inertia')) {
                return response()->json(['message' => 'Sự kiện đã hết hạn lưu trữ.'], 403);
            }
            return Inertia::render('Dashboard/Events/Expired', [
                'event' => $event->only('id', 'slug', 'title', 'expires_at'),
            ])->toResponse($request)->setStatusCode(403);
        }

        return $next($request);
    }
}
```

Apply vào dashboard event routes:

```php
Route::middleware(['auth', 'verified', 'role:host', EnsureEventNotExpired::class])
    ->prefix('dashboard/events/{event:slug}')
    ->group(function () {
        // tất cả dashboard event routes
    });
```

## Edge Cases

| Tình huống | Xử lý |
|-----------|-------|
| Import 50 dòng nhưng chỉ còn 30 slot | `assertCanAddGuests($event, 50)` throw trước khi import — không import partial |
| Nâng gói rồi downgrade (tương lai) | Không hỗ trợ downgrade ở MVP — plan chỉ tăng |
| Extra mua nhiều lần | `extra_guests` cộng dồn, không giới hạn số lần mua |
| Premium expires_at = null | `isExpired()` check `if (!$event->expires_at) return false` — không bao giờ expire |
| Free plan tạo lúc t=0, check 366 ngày sau | `expires_at->isPast()` đúng — event bị block |
| Host đang edit khi event expire | `EnsureEventNotExpired` middleware block mọi dashboard route khi expire |
| Quota check với extra: 200 basic + 100 extra = 300 | `max_guests = cfg['guests'] + event->extra_guests` |
| Concurrent guests import (2 tab cùng lúc) | Count-then-insert không atomic — tối đa overshoot 1 batch. Dùng DB constraint hoặc queue để strict nếu cần |
| Đếm rowCount import có thể sai nếu file có dòng trống | HeadingRowImport filter dòng trống — count dòng hợp lệ |
