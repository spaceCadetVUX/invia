# F2.4 Lời Chúc Dashboard

## Tổng quan

Host xem, pin, ẩn, xóa lời chúc. Pinned wishes hiển thị trên thiệp trước. Real-time polling 30s để cập nhật khi có lời chúc mới. Export PDF "Sổ lời chúc" để lưu làm kỷ niệm.

> **Note:** Routes + Controller đã định nghĩa trong F1.5. File này spec phần Vue dashboard + PDF export + polling.

## Controller — thêm exportPdf

```php
// app/Http/Controllers/Dashboard/DashboardWishController.php (bổ sung)

public function index(Event $event): Response
{
    $this->authorize('update', $event);

    return Inertia::render('Dashboard/Events/WishesDashboard', [
        'event'   => $event->only('id', 'slug', 'title'),
        'wishes'  => $this->buildQuery($event)->paginate(30),
        'summary' => $this->getSummary($event),
    ]);
}

public function exportPdf(Event $event): Response
{
    $this->authorize('update', $event);

    $wishes = Wish::where('event_id', $event->id)
        ->where('is_hidden', false)
        ->with('guest:id,name')
        ->orderByDesc('is_pinned')
        ->orderBy('created_at')
        ->get();

    $pdf = Pdf::loadView('pdf.wishes-book', [
        'event'  => $event,
        'wishes' => $wishes,
    ]);

    return $pdf->download("so-loi-chuc-{$event->slug}.pdf");
}

private function buildQuery(Event $event, array $filters = []): Builder
{
    $query = Wish::where('event_id', $event->id)
        ->with('guest:id,name,email')
        ->orderByDesc('is_pinned')
        ->orderByDesc('created_at');

    if (!empty($filters['search'])) {
        $query->where(function ($q) use ($filters) {
            $q->where('message', 'ilike', '%' . $filters['search'] . '%')
              ->orWhereHas('guest', fn ($g) =>
                  $g->where('name', 'ilike', '%' . $filters['search'] . '%')
              );
        });
    }

    if (isset($filters['show_hidden']) && $filters['show_hidden']) {
        // Host chọn hiện cả lời chúc đã ẩn
    } else {
        // Mặc định vẫn show tất cả trong dashboard (kể cả hidden — để host manage)
    }

    return $query;
}

private function getSummary(Event $event): array
{
    return [
        'total'   => Wish::where('event_id', $event->id)->count(),
        'pinned'  => Wish::where('event_id', $event->id)->where('is_pinned', true)->count(),
        'hidden'  => Wish::where('event_id', $event->id)->where('is_hidden', true)->count(),
        'visible' => Wish::where('event_id', $event->id)->where('is_hidden', false)->count(),
    ];
}
```

## Vue Component

```vue
<!-- resources/js/Pages/Dashboard/Events/WishesDashboard.vue -->
<script setup>
import { ref, onMounted, onUnmounted } from 'vue'
import { router } from '@inertiajs/vue3'
import axios from 'axios'

const props = defineProps({
    event:   Object,
    wishes:  Object,   // paginated
    summary: Object,
})

const summary = ref(props.summary)
const search  = ref('')
let   pollTimer = null

// Search
function applySearch() {
    router.get(
        route('dashboard.events.wishes.index', props.event.slug),
        { search: search.value },
        { preserveScroll: true, replace: true }
    )
}

// Polling mỗi 30s — chỉ reload summary + trang 1
function startPolling() {
    pollTimer = setInterval(() => {
        router.reload({
            only: ['summary', 'wishes'],
            preserveScroll: true,
            preserveState: true,
        })
    }, 30000)
}

// Toggle pin
function togglePin(wish) {
    router.patch(
        route('dashboard.events.wishes.pin', [props.event.slug, wish.id]),
        {},
        { preserveScroll: true, preserveState: true }
    )
}

// Toggle hide
function toggleHide(wish) {
    router.patch(
        route('dashboard.events.wishes.hide', [props.event.slug, wish.id]),
        {},
        { preserveScroll: true, preserveState: true }
    )
}

// Delete
function deleteWish(wish) {
    if (!confirm(`Xóa lời chúc của ${wish.guest.name}?`)) return
    router.delete(
        route('dashboard.events.wishes.destroy', [props.event.slug, wish.id]),
        { preserveScroll: true }
    )
}

onMounted(startPolling)
onUnmounted(() => clearInterval(pollTimer))
</script>

<template>
    <DashboardLayout>
        <!-- Summary -->
        <div class="grid grid-cols-4 gap-3 mb-6">
            <StatCard label="Tổng"       :value="summary.total" />
            <StatCard label="Đang hiện"  :value="summary.visible" color="green" />
            <StatCard label="Đã ghim"    :value="summary.pinned"  color="yellow" />
            <StatCard label="Đã ẩn"      :value="summary.hidden"  color="gray" />
        </div>

        <!-- Toolbar -->
        <div class="flex gap-2 mb-4">
            <input v-model="search" @keyup.enter="applySearch"
                   placeholder="Tìm tên hoặc nội dung..." class="input-field w-72" />
            <button @click="applySearch" class="btn-primary">Tìm</button>

            <a :href="route('dashboard.events.wishes.export-pdf', event.slug)"
               class="btn-secondary ml-auto">
                ↓ Export PDF Sổ lời chúc
            </a>
        </div>

        <!-- Wish list -->
        <div class="space-y-3">
            <div v-for="wish in wishes.data" :key="wish.id"
                 class="bg-white border rounded-lg p-4 flex gap-3"
                 :class="{
                     'border-yellow-300 bg-yellow-50': wish.is_pinned,
                     'opacity-50':                     wish.is_hidden,
                 }">

                <!-- Avatar -->

                    {{ wish.guest.name[0].toUpperCase() }}
                </div>

                <!-- Content -->
                <div class="flex-1 min-w-0">
                    <div class="flex items-baseline gap-2">
                        <span class="font-medium text-sm">{{ wish.guest.name }}</span>
                        <span v-if="wish.is_pinned" class="text-xs text-yellow-600">📌 Đã ghim</span>
                        <span v-if="wish.is_hidden" class="text-xs text-gray-400">🚫 Đã ẩn</span>
                        <time class="text-xs text-gray-400 ml-auto">{{ wish.created_at }}</time>
                    </div>
                    <p class="text-sm text-gray-700 mt-1 whitespace-pre-line">{{ wish.message }}</p>

                <div class="flex flex-col gap-1 flex-shrink-0">
                    <button @click="togglePin(wish)"
                            class="btn-icon text-xs"
                            :class="wish.is_pinned ? 'text-yellow-500' : 'text-gray-400'"
                            :title="wish.is_pinned ? 'Bỏ ghim' : 'Ghim lên đầu'">
                        📌
                    </button>
                    <button @click="toggleHide(wish)"
                            class="btn-icon text-xs"
                            :class="wish.is_hidden ? 'text-blue-500' : 'text-gray-400'"
                            :title="wish.is_hidden ? 'Hiện lại' : 'Ẩn khỏi thiệp'">
```

## Khi Inertia patch trả về updated wish

`DashboardWishController::pin()` và `hide()` trả JSON. Vue cần update local state để UI phản hồi ngay mà không reload trang:

```js
// Dùng router.patch với preserveState để Inertia tự merge props
router.patch(
    route('dashboard.events.wishes.pin', [props.event.slug, wish.id]),
    {},
    {
        preserveScroll: true,
        preserveState: true,
        onSuccess: () => {
            // Inertia tự reload 'wishes' prop sau patch
        },
    }
)
```

Hoặc nếu muốn instant update không chờ reload:

```js
// Optimistic update
wish.is_pinned = !wish.is_pinned
// Sau đó gọi PATCH — nếu lỗi rollback
```

MVP dùng `preserveState: true` + Inertia reload — đơn giản hơn.

## Edge Cases

| Tình huống | Xử lý |
|-----------|-------|
| Pin > 1 wish | Không giới hạn — host tự quyết định ghim bao nhiêu |
| PDF nhiều trang | dompdf tự paginate, `page-break-inside: avoid` tránh cắt giữa lời chúc |
| PDF font tiếng Việt | Dompdf cần font hỗ trợ UTF-8 — dùng `DejaVu Serif` fallback nếu Google Font không load |
| Wish bị ẩn nhưng vẫn show trong dashboard | Dashboard show tất cả (kể cả hidden) để host manage. `is_hidden` chỉ filter phía thiệp public |
| Search cross-field | `ilike` trên cả `message` và `guest.name` trong 1 query |
| 1000+ wishes export PDF | Timeout dompdf — cân nhắc queue job cho PDF, download async |
| Xóa wish có guest `self_register` | Wish bị xóa, guest record vẫn còn (có thể có RSVP) |
| Poll reload khi host đang type vào search | `preserveState: true` — search input không bị reset |
