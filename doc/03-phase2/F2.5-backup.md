# F2.5 Backup & Export

## Tổng quan

Host tạo backup toàn bộ sự kiện thành 1 file `.zip` gồm 6 file. Việc generate chạy qua queue job (không block UI). Khi xong, host nhận email + notification với link download có TTL 24h.

---

## 6 File Trong Zip

| File | Nội dung | Tạo bằng |
|------|---------|---------|
| `guests.xlsx` | Danh sách khách + RSVP + số bàn | maatwebsite/excel |
| `wishes.xlsx` | Lời chúc dạng bảng | maatwebsite/excel |
| `event-summary.pdf` | Tóm tắt sự kiện layout đẹp | dompdf |
| `wishes-book.pdf` | Sách lời chúc (như F2.4 export) | dompdf |
| `thiep.pdf` | Thiệp cưới bản in A4 | dompdf |
| `thiep.html` | Thiệp responsive tự chứa — xem offline, có animation | PHP render + inline |

## DB — Bảng event_backups

```sql
CREATE TABLE event_backups (
    id         BIGSERIAL PRIMARY KEY,
    event_id   BIGINT NOT NULL REFERENCES events(id) ON DELETE CASCADE,
    token      VARCHAR(64) NOT NULL UNIQUE,
    zip_path   VARCHAR(400),           -- null khi đang generate
    status     VARCHAR(20) NOT NULL DEFAULT 'pending',  -- pending | ready | failed
    expires_at TIMESTAMPTZ,            -- set khi ready, now()+24h
    created_at TIMESTAMPTZ NOT NULL DEFAULT now()
);
```

## Controller

```php
// app/Http/Controllers/Dashboard/BackupController.php
class BackupController extends Controller
{
    public function create(Event $event): JsonResponse
    {
        $this->authorize('update', $event);

        // Tránh duplicate — nếu đang pending/ready chưa hết hạn thì không tạo lại
        $existing = EventBackup::where('event_id', $event->id)
            ->whereIn('status', ['pending', 'ready'])
            ->where(fn ($q) => $q->whereNull('expires_at')->orWhere('expires_at', '>', now()))
            ->latest()
            ->first();

        if ($existing?->status === 'ready') {
            return response()->json([
                'status'       => 'ready',
                'download_url' => route('dashboard.events.backup.download', [$event->slug, $existing->token]),
                'expires_at'   => $existing->expires_at,
            ]);
        }

        if ($existing?->status === 'pending') {
            return response()->json(['status' => 'pending', 'message' => __('backup.already_processing')]);
        }

        $backup = EventBackup::create([
            'event_id' => $event->id,
            'token'    => Str::random(48),
            'status'   => 'pending',
        ]);

        GenerateBackupJob::dispatch($backup);

        return response()->json(['status' => 'pending', 'message' => __('backup.queued')], 202);
    }

    public function status(Event $event): JsonResponse
    {
        $this->authorize('update', $event);

        $backup = EventBackup::where('event_id', $event->id)
            ->latest()
            ->first();

        if (!$backup) {
            return response()->json(['status' => 'none']);
        }

        return response()->json([
            'status'       => $backup->status,
            'download_url' => $backup->status === 'ready'
                ? route('dashboard.events.backup.download', [$event->slug, $backup->token])
                : null,
            'expires_at'   => $backup->expires_at,
        ]);
    }

    public function download(Event $event, string $token): StreamedResponse
    {
        $this->authorize('update', $event);

        $backup = EventBackup::where('event_id', $event->id)
            ->where('token', $token)
            ->where('status', 'ready')
            ->firstOrFail();

        abort_if($backup->expires_at && $backup->expires_at->isPast(), 410, __('backup.link_expired'));

        return Storage::disk()->download($backup->zip_path, "invia-backup-{$event->slug}.zip");
    }
}
```

## Queue Job

```php
// app/Jobs/GenerateBackupJob.php
class GenerateBackupJob implements ShouldQueue
{
    public int $tries   = 2;
    public int $timeout = 120;   // 2 phút max

    public function __construct(public EventBackup $backup) {}

    public function handle(BackupService $service): void
    {
        $event = $this->backup->event()->with(['template', 'user'])->firstOrFail();

        try {
            $zipPath = $service->generate($event, $this->backup->token);

            $this->backup->update([
                'status'     => 'ready',
                'zip_path'   => $zipPath,
                'expires_at' => now()->addHours(24),
            ]);

            // Notify host
            app(NotificationService::class)->notifyHost($event, 'system', [
                'title'        => __('backup.ready_title'),
                'download_url' => route('dashboard.events.backup.download', [$event->slug, $this->backup->token]),
                'event_id'     => $event->id,
            ]);

            Mail::to($event->user->email)->send(new BackupReadyMail($event, $this->backup));

        } catch (\Throwable $e) {
            $this->backup->update(['status' => 'failed']);
            Log::error("Backup failed event {$event->id}: " . $e->getMessage());
            throw $e;
        }
    }
}
```

## BackupService — Generate 6 files + Zip

```php
// app/Services/BackupService.php
class BackupService
{
    public function generate(Event $event, string $token): string
    {
        $tmpDir = sys_get_temp_dir() . "/backup-{$token}";
        mkdir($tmpDir, 0755, true);

        try {
            $this->writeGuestsExcel($event, $tmpDir);
            $this->writeWishesExcel($event, $tmpDir);
            $this->writeEventSummaryPdf($event, $tmpDir);
            $this->writeWishesBookPdf($event, $tmpDir);
            $this->writeThiepPdf($event, $tmpDir);
            $this->writeThiepHtml($event, $tmpDir);

            $zipPath = $this->createZip($event, $token, $tmpDir);

            return $zipPath;
        } finally {
            // Dọn temp dir dù success hay fail
            $this->rrmdir($tmpDir);
        }
    }

    // ── 1. guests.xlsx ────────────────────────────────────
    private function writeGuestsExcel(Event $event, string $dir): void
    {
        Excel::store(new GuestsExport($event), $dir . '/guests.xlsx', 'local_abs');
    }

    // ── 2. wishes.xlsx ────────────────────────────────────
    private function writeWishesExcel(Event $event, string $dir): void
    {
        Excel::store(new WishesExport($event), $dir . '/wishes.xlsx', 'local_abs');
    }

    // ── 3. event-summary.pdf ──────────────────────────────
    private function writeEventSummaryPdf(Event $event, string $dir): void
    {
        $rsvpStats = [
            'yes'   => Rsvp::where('event_id', $event->id)->where('status', 'yes')->count(),
            'no'    => Rsvp::where('event_id', $event->id)->where('status', 'no')->count(),
            'maybe' => Rsvp::where('event_id', $event->id)->where('status', 'maybe')->count(),
            'total' => Guest::where('event_id', $event->id)->count(),
        ];

        Pdf::loadView('pdf.event-summary', compact('event', 'rsvpStats'))
            ->save($dir . '/event-summary.pdf');
    }

    // ── 4. wishes-book.pdf ────────────────────────────────
    private function writeWishesBookPdf(Event $event, string $dir): void
    {
        $wishes = Wish::where('event_id', $event->id)
            ->where('is_hidden', false)
            ->with('guest:id,name')
            ->orderByDesc('is_pinned')
            ->orderBy('created_at')
            ->get();

        Pdf::loadView('pdf.wishes-book', compact('event', 'wishes'))
            ->save($dir . '/wishes-book.pdf');
    }

    // ── 5. thiep.pdf ──────────────────────────────────────
    private function writeThiepPdf(Event $event, string $dir): void
    {
        Pdf::loadView('pdf.thiep', compact('event'))
            ->setPaper('a4', 'portrait')
            ->save($dir . '/thiep.pdf');
    }

    // ── 6. thiep.html — self-contained ───────────────────
    private function writeThiepHtml(Event $event, string $dir): void
    {
        // Render Blade template
        $templateHtml = view("templates.{$event->template->blade_file}.index", [
            'event'     => $event,
            'guest'     => null,
            'music'     => ['type' => 'none'],
            'ogMeta'    => [],
            'isPreview' => false,
        ])->render();

        // Đọc CSS của template
        $cssPath = public_path("templates/{$event->template->blade_file}/style.css");
        $css     = file_exists($cssPath) ? file_get_contents($cssPath) : '';

        // Đọc JS của template
        $jsPath  = public_path("templates/{$event->template->blade_file}/script.js");
        $js      = file_exists($jsPath) ? file_get_contents($jsPath) : '';

        // Inline GSAP từ CDN — nếu offline, animation không chạy nhưng HTML vẫn hiện
        $gsapCdn = 'https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js';

        // Inject inline CSS + JS vào HTML
        $selfContained = <<<HTML
<!DOCTYPE html>
<html lang="vi">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>{$event->title} — Thiệp mời</title>
<style>{$css}</style>
</head>
<body>
{$templateHtml}
<script src="{$gsapCdn}"></script>
<script>{$js}</script>
</body>
</html>
HTML;

        file_put_contents($dir . '/thiep.html', $selfContained);
    }

    // ── Zip ───────────────────────────────────────────────
    private function createZip(Event $event, string $token, string $dir): string
    {
        $zipName = "backups/{$event->id}-{$token}.zip";
        $tmpZip  = sys_get_temp_dir() . "/{$token}.zip";

        $zip = new ZipArchive();
        $zip->open($tmpZip, ZipArchive::CREATE | ZipArchive::OVERWRITE);

        foreach (glob($dir . '/*') as $file) {
            $zip->addFile($file, basename($file));
        }

        $zip->close();

        // Upload lên Storage (local hay R2)
        Storage::put($zipName, file_get_contents($tmpZip));
        unlink($tmpZip);

        return $zipName;
    }

    private function rrmdir(string $dir): void
    {
        foreach (glob($dir . '/*') as $f) { unlink($f); }
        rmdir($dir);
    }
}
```

```php
// app/Exports/WishesExport.php
class WishesExport implements FromCollection, WithHeadings
{
    public function __construct(private Event $event) {}

    public function collection(): Collection
    {
        return Wish::where('event_id', $this->event->id)
            ->where('is_hidden', false)
            ->with('guest:id,name')
            ->orderByDesc('is_pinned')
            ->orderBy('created_at')
            ->get()
            ->map(fn ($w) => [
                $w->guest->name,
                $w->message,
                $w->is_pinned ? 'Có' : '',
                $w->created_at->format('d/m/Y H:i'),
            ]);
    }

    public function headings(): array
    {
        return ['Tên khách', 'Lời chúc', 'Nổi bật', 'Thời gian'];
    }
}
```

## Vue Component (Dashboard trigger + status)

```vue
<!-- resources/js/Components/BackupPanel.vue -->
<script setup>
import { ref, onMounted } from 'vue'
import axios from 'axios'

const props   = defineProps({ event: Object })
const status  = ref('idle')   // idle | pending | ready | failed | expired
const downloadUrl = ref(null)
const expiresAt   = ref(null)
let   pollTimer   = null

async function checkStatus() {
    const res = await axios.get(route('dashboard.events.backup.status', props.event.slug))
    status.value      = res.data.status ?? 'idle'
    downloadUrl.value = res.data.download_url
    expiresAt.value   = res.data.expires_at

    if (status.value === 'pending') {
        pollTimer = setTimeout(checkStatus, 5000)   // poll mỗi 5s khi đang generate
    }
}

async function createBackup() {
    status.value = 'pending'
    await axios.post(route('dashboard.events.backup.create', props.event.slug))
    pollTimer = setTimeout(checkStatus, 5000)
}

onMounted(checkStatus)
</script>

<template>
    <div class="backup-panel border rounded-lg p-4">
        <h3 class="font-medium mb-3">Tải backup sự kiện</h3>

        <p class="text-sm text-gray-500 mb-4">
            1 file .zip gồm: danh sách khách (Excel), lời chúc (Excel + PDF),
            tóm tắt sự kiện (PDF), thiệp cưới (PDF + HTML offline).
        </p>

        <!-- Idle / Failed -->
        <button v-if="status === 'idle' || status === 'failed'"
                @click="createBackup"
                class="btn-primary">
            {{ status === 'failed' ? '↻ Thử lại' : 'Tạo backup' }}
        </button>

        <!-- Pending -->
        <div v-else-if="status === 'pending'" class="flex items-center gap-2 text-sm text-gray-500">
            <Spinner class="w-4 h-4" />
            Đang tạo backup... (khoảng 1–2 phút)
        </div>

        <!-- Ready -->
        <div v-else-if="status === 'ready'" class="space-y-2">
            <a :href="downloadUrl" class="btn-primary inline-flex items-center gap-2">
                ↓ Tải backup (.zip)
            </a>
            <p class="text-xs text-gray-400">
                Hết hạn: {{ new Date(expiresAt).toLocaleString('vi-VN') }}

            <button @click="createBackup" class="text-xs text-blue-500 underline">
                Tạo lại backup mới
            </button>
        </div>

            Link đã hết hạn.
            <button @click="createBackup" class="text-blue-500 underline">Tạo lại</button>
```

```php
// Schedule: daily

$expired = EventBackup::where('status', 'ready')
    ->where('expires_at', '<', now())
    ->get();

    if ($backup->zip_path) {
        Storage::delete($backup->zip_path);
    }
    $backup->update(['status' => 'expired', 'zip_path' => null]);
```

## Edge Cases

| Tình huống | Xử lý |
|-----------|-------|
| Job timeout > 2 phút | `$timeout = 120`, job fail → `status = failed`, host có thể retry |
| Event không có wishes | `wishes.xlsx` rỗng (chỉ có heading), `wishes-book.pdf` có trang bìa nhưng không có nội dung |
| Template CSS/JS file không tồn tại | `file_exists()` check trước khi đọc — fallback empty string |
| Storage disk full | Job fail → `status = failed`, log error |
| Host tạo backup 2 lần liên tiếp | Check existing `pending` — trả về "đang xử lý" thay vì tạo job mới |
| Link download sau 24h | `expires_at->isPast()` → HTTP 410, frontend show "tạo lại" |
| `thiep.html` font Google Fonts offline | CSS dùng `font-family` system fallback khi không có network |
| Zip file > 100MB (nhiều ảnh) | Upload thẳng lên Storage — không giới hạn ở job, giới hạn ở Storage quota |
| `on delete cascade` khi xóa event | `event_backups` xóa theo, Storage file cần xóa riêng (cleanup job) |
