# F2.3 Theo Dõi RSVP

## Tổng quan

Dashboard xem toàn bộ RSVP của event: filter theo trạng thái, search theo tên, sort, export. Hiển thị summary stats và breakdown theo bàn tiệc. Polling 30s để cập nhật real-time khi khách mới RSVP.

## Routes

```php
Route::middleware(['auth', 'verified', 'role:host'])

        Route::patch('/events/{event:slug}/rsvp/{rsvp}/table', [RsvpDashboardController::class, 'assignTable'])->name('events.rsvp.table');
```

## Controller

```php
// app/Http/Controllers/Dashboard/RsvpDashboardController.php
class RsvpDashboardController extends Controller
{
    public function __construct(private RsvpDashboardService $service) {}

    public function index(Event $event, Request $request): Response
    {
        $this->authorize('update', $event);

        $filters = $request->only(['status', 'search', 'sort', 'direction']);

        return Inertia::render('Dashboard/Events/RsvpDashboard', [
            'event'   => $event->only('id', 'slug', 'title', 'event_date'),
            'rsvps'   => $this->service->getFiltered($event, $filters),
            'summary' => $this->service->getSummary($event),
            'filters' => $filters,
        ]);
    }

    public function export(Event $event): BinaryFileResponse
    {
        $this->authorize('update', $event);

        return Excel::download(new RsvpExport($event), "rsvp-{$event->slug}.xlsx");
    }

    public function assignTable(Request $request, Event $event, Rsvp $rsvp): JsonResponse
    {
        $this->authorize('update', $event);
        abort_if($rsvp->event_id !== $event->id, 403);

        $data = $request->validate([
            'table_no' => ['nullable', 'string', 'max:20'],
        ]);

        // Cập nhật table_no trên guest record (không phải RSVP)
        $rsvp->guest->update(['table_no' => $data['table_no']]);

        return response()->json(['table_no' => $data['table_no']]);
    }
}
```

## Service

```php
// app/Services/RsvpDashboardService.php
class RsvpDashboardService
{
    public function getFiltered(Event $event, array $filters): LengthAwarePaginator
    {
        $query = Rsvp::where('event_id', $event->id)
            ->with('guest:id,name,email,phone,table_no')
            ->select('rsvp.*');

        // Filter theo status
        if (!empty($filters['status']) && in_array($filters['status'], ['yes', 'no', 'maybe'])) {
            $query->where('status', $filters['status']);
        }

        // Search theo tên khách
        if (!empty($filters['search'])) {
            $query->whereHas('guest', fn ($q) =>
                $q->where('name', 'ilike', '%' . $filters['search'] . '%')
            );
        }

        // Sort
        $sortCol = match ($filters['sort'] ?? 'created_at') {
            'name'       => 'guests.name',
            'status'     => 'rsvp.status',
            'plus_one'   => 'rsvp.plus_one',
            'created_at' => 'rsvp.created_at',
            default      => 'rsvp.created_at',
        };

        $dir = ($filters['direction'] ?? 'desc') === 'asc' ? 'asc' : 'desc';

        if ($sortCol === 'guests.name') {
            $query->join('guests', 'rsvp.guest_id', '=', 'guests.id')->orderBy($sortCol, $dir);
        } else {
            $query->orderBy($sortCol, $dir);
        }

        return $query->paginate(50)->withQueryString();
    }

    public function getSummary(Event $event): array
    {
        $rows = Rsvp::where('event_id', $event->id)
            ->selectRaw('status, COUNT(*) as count, SUM(plus_one) as plus_ones')
            ->groupBy('status')
            ->get()
            ->keyBy('status');

        $yes   = $rows['yes']   ?? null;
        $no    = $rows['no']    ?? null;
        $maybe = $rows['maybe'] ?? null;

        $totalGuests = ($yes?->count   ?? 0)
                     + ($maybe?->count ?? 0);  // không tính 'no'

        $totalPeople = $totalGuests
                     + ($yes?->plus_ones   ?? 0)
                     + ($maybe?->plus_ones ?? 0);

        return [
            'yes'          => (int) ($yes?->count   ?? 0),
            'no'           => (int) ($no?->count    ?? 0),
            'maybe'        => (int) ($maybe?->count ?? 0),
            'total_guests' => $totalGuests,
            'total_people' => $totalPeople,    // bao gồm plus_one
            'updated_at'   => now()->toISOString(),
        ];
    }
}
```

## Export Excel

```php
// app/Exports/RsvpExport.php
class RsvpExport implements FromCollection, WithHeadings
{
    public function __construct(private Event $event) {}

    public function collection(): Collection
    {
        return Rsvp::where('event_id', $this->event->id)
            ->with('guest:id,name,email,phone,table_no')
            ->orderBy('status')
            ->get()
            ->map(fn ($r) => [
                $r->guest->name,
                $r->guest->email,
                $r->guest->phone,
                match ($r->status) {
                    'yes'   => 'Tham dự',
                    'no'    => 'Từ chối',
                    'maybe' => 'Có thể',
                    default => '—',
                },
                $r->plus_one ?? 0,
                $r->dietary,
                $r->guest->table_no,
                $r->created_at->format('d/m/Y H:i'),
            ]);
    }

    public function headings(): array
    {
        return ['Tên', 'Email', 'SĐT', 'Tham dự', 'Người đi cùng', 'Yêu cầu ăn uống', 'Bàn', 'Thời gian RSVP'];
    }
}
```

```vue
const statusFilter = ref(props.filters.status ?? '')
let   pollTimer  = null

// Filter/search gửi qua Inertia GET (giữ URL đẹp, có thể share/bookmark)
function applyFilters() {
    router.get(
        route('dashboard.events.rsvp.index', props.event.slug),
        { status: statusFilter.value, search: search.value },
        { preserveScroll: true, replace: true }

// Sort khi click header
function sortBy(col) {
    const currentDir = props.filters.direction ?? 'desc'
    router.get(
        route('dashboard.events.rsvp.index', props.event.slug),
        {
            ...props.filters,
            sort:      col,
            direction: props.filters.sort === col && currentDir === 'asc' ? 'desc' : 'asc',
        },
        { preserveScroll: true, replace: true }
    )
}

    pollTimer = setInterval(() => {
        router.reload({
            only: ['summary'],
            preserveScroll: true,
            preserveState: true,

    }, 30000)

onUnmounted(() => clearInterval(pollTimer))
```

## TableNoInput Component (inline edit bàn)

```vue
<!-- resources/js/Components/TableNoInput.vue -->
<script setup>
import { ref } from 'vue'
import axios from 'axios'

const props = defineProps({ rsvp: Object, event: Object })
const editing  = ref(false)
const tableVal = ref(props.rsvp.guest.table_no ?? '')
const saving   = ref(false)

async function save() {
    saving.value = true
    await axios.patch(
        route('dashboard.events.rsvp.table', [props.event.slug, props.rsvp.id]),
        { table_no: tableVal.value || null }
    )
    saving.value = false
    editing.value = false
}
</script>

<template>
    <span v-if="!editing" @click="editing = true"
          class="cursor-pointer text-sm text-gray-600 hover:text-blue-600 min-w-[40px] inline-block">
        {{ tableVal || '—' }}
    </span>
    <input v-else v-model="tableVal" @blur="save" @keyup.enter="save" @keyup.esc="editing = false"
           :disabled="saving" v-focus
           class="border rounded px-1 py-0.5 text-sm w-16" />
</template>
```

## Inertia Partial Reload (polling summary)

Polling dùng Inertia partial reload để chỉ fetch `summary` prop, không reload toàn trang:

```js
router.reload({
    only: ['summary'],
    preserveScroll: true,
    preserveState: true,
})
```

Thay `axios.get` bằng `router.reload` để tận dụng Inertia protocol. Đơn giản hơn và không cần parse response thủ công.

## Edge Cases

| Tình huống | Xử lý |
|-----------|-------|
| Search tên có dấu (`ilike`) | PostgreSQL `ilike` case-insensitive, hỗ trợ UTF-8 dấu tiếng Việt |
| Sort theo `guests.name` join không an toàn | Chỉ join khi `sort = 'name'` — không join mặc định |
| `plus_one` null trong `total_people` | `SUM(plus_one)` trả NULL nếu tất cả null → dùng `?? 0` |
| RSVP của guest bị xóa (orphaned) | `ON DELETE CASCADE` — không có RSVP orphaned |
| `assignTable` cho RSVP event khác | `abort_if($rsvp->event_id !== $event->id, 403)` |
| 500+ RSVP cùng lúc xem | Paginate 50 / trang + index trên `(event_id, status)` |
| Export file lớn (1000+ rows) | `FromCollection` stream-friendly, không OOM |

## Checklist

- [ ] `ilike` dùng đúng — PostgreSQL only (không dùng `like` viết thường)
- [ ] Sort `guests.name` join chỉ khi `sort = 'name'` — tránh join không cần thiết
- [ ] `getSummary` tính `total_people` bao gồm `plus_one` của `yes` + `maybe`
- [ ] Polling dùng `router.reload({ only: ['summary'] })` — không phải axios
- [ ] `assignTable` update `guests.table_no`, không phải `rsvp` table
- [ ] Export bao gồm cột `dietary` và `table_no`
- [ ] DB index `(event_id, status)` và `(event_id, created_at DESC)`
- [ ] `withQueryString()` trên paginator để giữ filter params qua trang
