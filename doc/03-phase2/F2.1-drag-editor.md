# F2.1 Drag Editor

## Tổng quan

Host kéo thả các text slot trực tiếp trên canvas thiệp, chỉnh font/màu/size trong sidebar, auto-save debounced vào `events.settings` JSONB. Preview iframe reload sau mỗi lần save.

## Controller

```php
// app/Http/Controllers/Dashboard/EventEditorController.php
class EventEditorController extends Controller
{
    public function show(Event $event): Response
    {
        $this->authorize('update', $event);

        $configPath = public_path("templates/{$event->template->blade_file}/config.json");
        abort_unless(file_exists($configPath), 500, 'Template config.json missing');

        $templateConfig = json_decode(file_get_contents($configPath), true);

        return Inertia::render('Dashboard/Events/Editor', [
            'event'          => $event->only('id', 'slug', 'title', 'settings'),
            'templateConfig' => $templateConfig,
            'templateMeta'   => $event->template->only('blade_file', 'version'),
        ]);
    }

    public function saveSettings(SaveSettingsRequest $request, Event $event): JsonResponse
    {
        $this->authorize('update', $event);

        $event->update(['settings' => $request->validated('settings')]);

        // Invalidate public page cache
        Cache::forget("invitation:{$event->slug}");

        return response()->json(['saved_at' => now()->toISOString()]);
    }

    public function preview(Event $event): View
    {
        $this->authorize('update', $event);

        // Cho phép preview kể cả draft — chỉ host mới truy cập được route này
        return view("templates.{$event->template->blade_file}.index", [
            'event'     => $event,
            'guest'     => null,
            'music'     => ['type' => 'none'],   // tắt nhạc trong preview
            'ogMeta'    => [],
            'isPreview' => true,
        ]);
    }
}
```

## Request Validation

```php
// app/Http/Requests/SaveSettingsRequest.php
class SaveSettingsRequest extends FormRequest
{
    public function rules(): array
    {
        return [
            'settings'                  => ['required', 'array'],
            'settings.*'                => ['array'],
            'settings.*.x'              => ['nullable', 'numeric', 'min:0', 'max:100'],
            'settings.*.y'              => ['nullable', 'numeric', 'min:0', 'max:100'],
            'settings.*.font'           => ['nullable', 'string', 'max:80'],
            'settings.*.size'           => ['nullable', 'integer', 'min:8', 'max:200'],
            'settings.*.color'          => ['nullable', 'string', 'regex:/^#[0-9a-fA-F]{3,6}$/'],
            'settings.*.value'          => ['nullable', 'string', 'max:300'],
        ];
    }
}
```

## events.settings JSONB structure

```json
{
  "bride_name": { "x": 48, "y": 32, "font": "Playfair Display",    "size": 48, "color": "#ffffff", "value": "Nguyễn Thị Lan" },
  "groom_name": { "x": 52, "y": 42, "font": "Playfair Display",    "size": 48, "color": "#ffffff", "value": "Trần Văn Minh"  },
  "date":       { "x": 50, "y": 56, "font": "Cormorant Garamond",  "size": 24, "color": "#eeeeee", "value": null }
}
```

`value` null → slot dùng field tương ứng của event (`event.title`, `event.event_date`…). Set `value` khi host muốn override text mặc định.

## Vue Component

```vue
<!-- resources/js/Pages/Dashboard/Events/Editor.vue -->
<script setup>
import { ref, reactive, computed, watch, onMounted, nextTick } from 'vue'
import { useDraggable, useElementSize } from '@vueuse/core'
import { useDebounceFn } from '@vueuse/core'
import axios from 'axios'
import { route } from 'ziggy-js'

const props = defineProps({
    event:          Object,
    templateConfig: Object,
    templateMeta:   Object,
})

// ── State ──────────────────────────────────────────────
const canvasRef    = ref(null)
const { width: canvasW, height: canvasH } = useElementSize(canvasRef)
const saveStatus   = ref('idle')   // 'idle' | 'saving' | 'saved' | 'error'
const selectedSlot = ref(null)
const previewKey   = ref(0)        // increment để force iframe reload

// Khởi tạo settings từ event.settings + fallback về default từ config.json
const settings = reactive(
    Object.fromEntries(
        Object.entries(props.templateConfig.slots).map(([key, def]) => [
            key,
            {
                x:     props.event.settings[key]?.x     ?? def.default_x,
                y:     props.event.settings[key]?.y     ?? def.default_y,
                font:  props.event.settings[key]?.font  ?? props.templateConfig.fonts[0],
                size:  props.event.settings[key]?.size  ?? 32,
                color: props.event.settings[key]?.color ?? '#000000',
                value: props.event.settings[key]?.value ?? null,
            }
        ])
    )
)

// ── useDraggable per slot ───────────────────────────────
// Tạo ref map cho từng slot element
const slotRefs = reactive({})

// useDraggable phải gọi trong setup context (không phải event handler)
onMounted(async () => {
    await nextTick()   // đảm bảo slotRefs đã populated
    Object.keys(props.templateConfig.slots).forEach(key => initSlotDrag(key))
})

function initSlotDrag(key) {
    if (!slotRefs[key]) return
    const { x: px, y: py } = useDraggable(slotRefs[key], {
        initialValue: {
            x: (settings[key].x / 100) * canvasW.value,
            y: (settings[key].y / 100) * canvasH.value,
        },
        onEnd(pos) {
            // Chuyển về % để responsive
            settings[key].x = Math.round((pos.x / canvasW.value) * 100 * 10) / 10
            settings[key].y = Math.round((pos.y / canvasH.value) * 100 * 10) / 10
            triggerSave()
        },
    })
}

// ── Auto-save ──────────────────────────────────────────
const triggerSave = useDebounceFn(async () => {
    saveStatus.value = 'saving'
    try {
        await axios.patch(route('dashboard.events.settings.save', props.event.slug), {
            settings,
        })
        saveStatus.value = 'saved'
        previewKey.value++   // force iframe reload
    } catch {
        saveStatus.value = 'error'
    }
}, 1500)

// Watch sidebar changes → auto-save
watch(settings, triggerSave, { deep: true })

// ── Sidebar helpers ────────────────────────────────────
const selectedConfig = computed(() =>
    selectedSlot.value ? props.templateConfig.slots[selectedSlot.value] : null
)

const selectedSettings = computed(() =>
    selectedSlot.value ? settings[selectedSlot.value] : null
)
</script>

<template>
    <DashboardLayout>
        <!-- Toolbar -->
        <div class="editor-toolbar flex items-center justify-between px-4 py-2 border-b">
            <a :href="route('dashboard.events.show', event.slug)" class="btn-back">← Quay lại</a>
            <span class="save-status text-sm text-gray-500">
                <span v-if="saveStatus === 'saving'">Đang lưu...</span>
                <span v-else-if="saveStatus === 'saved'" class="text-green-600">Đã lưu ✓</span>
                <span v-else-if="saveStatus === 'error'" class="text-red-500">Lỗi lưu</span>
            </span>
            <a :href="route('invitation.show', event.slug)" target="_blank" class="btn-preview">
                Xem thiệp ↗
            </a>
        </div>

        <div class="editor-body flex h-[calc(100vh-56px)]">
            <!-- Canvas -->
            <div class="editor-canvas flex-1 relative overflow-hidden bg-gray-100 flex items-center justify-center">
                <!-- Template background (CSS từ template) -->
                <div ref="canvasRef"
                     class="canvas-inner relative"
                     :style="{ width: templateConfig.canvas.width + 'px', height: templateConfig.canvas.height + 'px' }">

                    <!-- Template background layer -->
                    <link :href="`/templates/${templateMeta.blade_file}/style.css?v=${templateMeta.version}`" rel="stylesheet">
                    <div :class="`tmpl-${templateMeta.blade_file} w-full h-full`"></div>

                    <!-- Draggable slot overlays -->
                    <div
                        v-for="(slotDef, key) in templateConfig.slots"
                        :key="key"
                        :ref="el => slotRefs[key] = el"
                        class="slot-handle absolute cursor-move select-none"
                        :class="{ 'ring-2 ring-blue-500': selectedSlot === key }"
                        :style="{
                            left:     settings[key].x + '%',
                            top:      settings[key].y + '%',
                            fontFamily: settings[key].font,
                            fontSize:   settings[key].size + 'px',
                            color:      settings[key].color,
                            transform: 'translate(-50%, -50%)',
                        }"
                        @click="selectedSlot = key"
                    >
                        <!-- Text slot -->
                        <template v-if="slotDef.type === 'text'">
                            {{ settings[key].value ?? slotDef.label }}
                        </template>
                        <!-- Image slot -->
                        <template v-else-if="slotDef.type === 'image'">
                            <div class="slot-image-placeholder w-24 h-24 border-2 border-dashed border-white flex items-center justify-center text-white text-xs">
                                {{ slotDef.label }}
                            </div>
                        </template>
                        <!-- Video slot -->
                        <template v-else-if="slotDef.type === 'video'">
                            <div class="slot-video-placeholder w-32 h-20 border-2 border-dashed border-white flex items-center justify-center text-white text-xs">
                                ▶ {{ slotDef.label }}
                            </div>
                        </template>
                    </div>
                </div>
            </div>

            <!-- Sidebar -->
            <aside class="editor-sidebar w-72 border-l bg-white overflow-y-auto flex flex-col">
                <!-- Slot list -->
                <div class="p-4 border-b">
                    <h3 class="text-xs uppercase text-gray-400 mb-2">Phần tử</h3>
                    <button
                        v-for="(slotDef, key) in templateConfig.slots"
                        :key="key"
                        @click="selectedSlot = key"
                        class="w-full text-left px-3 py-2 rounded text-sm"
                        :class="selectedSlot === key ? 'bg-blue-50 text-blue-700' : 'hover:bg-gray-50'"
                    >
                        {{ slotDef.label }}
                        <span class="text-xs text-gray-400 ml-1">({{ slotDef.type }})</span>
                    </button>
                </div>

                <!-- Controls cho slot đang chọn -->
                <div v-if="selectedSlot && selectedConfig?.type === 'text'" class="p-4 space-y-4">
                    <h3 class="text-xs uppercase text-gray-400">Tuỳ chỉnh</h3>

                    <!-- Custom value override -->
                    <div>
                        <label class="text-xs text-gray-600">Nội dung</label>
                        <input type="text" v-model="selectedSettings.value" placeholder="Để trống = dùng mặc định"
                               class="w-full border rounded px-2 py-1 text-sm mt-1">
                    </div>

                    <!-- Font -->
                    <div>
                        <label class="text-xs text-gray-600">Font chữ</label>
                        <select v-model="selectedSettings.font" class="w-full border rounded px-2 py-1 text-sm mt-1">
                            <option v-for="f in templateConfig.fonts" :key="f" :value="f">{{ f }}</option>
                        </select>
                    </div>

                    <!-- Size -->
                    <div>
                        <label class="text-xs text-gray-600">Cỡ chữ</label>
                        <div class="flex items-center gap-2 mt-1">
                            <input type="range" v-model.number="selectedSettings.size" min="8" max="120"
                                   class="flex-1">
                            <span class="text-sm w-10 text-center">{{ selectedSettings.size }}px</span>
                        </div>
                    </div>

                    <!-- Color -->
                    <div>
                        <label class="text-xs text-gray-600">Màu chữ</label>
                        <div class="flex items-center gap-2 mt-1">
                            <input type="color" v-model="selectedSettings.color" class="w-8 h-8 rounded cursor-pointer">
                            <input type="text" v-model="selectedSettings.color" maxlength="7"
                                   class="flex-1 border rounded px-2 py-1 text-sm font-mono">
                        </div>
                    </div>

                    <!-- Position (manual) -->
                    <div class="grid grid-cols-2 gap-2">
                        <div>
                            <label class="text-xs text-gray-600">X (%)</label>
                            <input type="number" v-model.number="selectedSettings.x" min="0" max="100"
                                   class="w-full border rounded px-2 py-1 text-sm mt-1">
                        </div>
                        <div>
                            <label class="text-xs text-gray-600">Y (%)</label>
                            <input type="number" v-model.number="selectedSettings.y" min="0" max="100"
                                   class="w-full border rounded px-2 py-1 text-sm mt-1">
                        </div>
                    </div>
                </div>

                <!-- Preview iframe -->
                <div class="p-4 border-t mt-auto">
                    <h3 class="text-xs uppercase text-gray-400 mb-2">Preview</h3>
                    <iframe
                        :key="previewKey"
                        :src="route('dashboard.events.preview', event.slug)"
                        class="w-full h-64 border rounded"
                        sandbox="allow-scripts allow-same-origin"
                    ></iframe>
                </div>
            </aside>
        </div>
    </DashboardLayout>
</template>
```

## Auto-save Flow

```
User drag slot / change sidebar
  → watch(settings, deep: true)
    → useDebounceFn 1.5s
      → PATCH /dashboard/events/{slug}/settings
        → SaveSettingsRequest validate
          → event.update(['settings' => ...])
          → Cache::forget("invitation:{slug}")
          → return { saved_at }
        → saveStatus = 'saved'
        → previewKey++ → iframe reload
```

## Edge Cases

| Tình huống | Xử lý |
|-----------|-------|
| `config.json` không tồn tại | `abort(500)` rõ ràng thay vì PHP fatal |
| Drag ra ngoài canvas | `useDraggable` tự giới hạn theo container — thêm `clamp` nếu cần |
| `x` hoặc `y` > 100 sau tính toán | Validation `max:100` ở server |
| `color` không đúng hex | Validation `regex:/^#[0-9a-fA-F]{3,6}$/` |
| Auto-save race condition (2 saves liên tiếp) | Debounce 1.5s — chỉ save lần cuối |
| Host đang edit, AI hết phiên | `settings` luôn sync sau mỗi save, không mất data |
| Template thay đổi (F1.2) reset settings | `settings = []` sau đổi template — editor load lại default |
| `value` null trong settings | Blade template hiển thị giá trị mặc định của slot |
| Preview iframe bị CSP block | `sandbox="allow-scripts allow-same-origin"` đủ cho template JS + CSS |
